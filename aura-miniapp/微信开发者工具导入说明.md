# 微信开发者工具导入路径

请确保在微信开发者工具中，**项目目录**指向以下路径之一：

- **开发模式**（`npm run dev:mp-weixin` 运行中）：  
  `aura-miniapp/dist/dev/mp-weixin`

- **构建模式**（`npm run build:mp-weixin` 后）：  
  `aura-miniapp/dist/build/mp-weixin`

⚠️ 不要选择 `aura-miniapp` 根目录，必须是 `dist/dev` 或 `dist/build` 下的 `mp-weixin` 文件夹。

## 若仍显示旧问候语

1. 在微信开发者工具中：**工具 → 清除缓存 → 清除全部**
2. 关闭项目后，重新选择上述正确路径导入
3. 点击「编译」重新编译
